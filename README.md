# ToAllTool

A comprehensive tool aggregator platform featuring 800+ tools across 40 categories. Discover, explore, and access the tools you need all in one place.

**Live:** [toalltool.online](https://toalltool.online)

## Overview

ToAllTool is a curated collection of over 800 web tools organized into 40 categories, making it easy to find exactly what you need. Whether you're looking for converters, generators, analyzers, or productivity tools — they're all here.

### Key Features

- 🛠️ **800+ Tools** across 40 categories
- 🎯 **Well-Organized** categories for quick navigation
- ⚡ **Fast & Responsive** built for speed
- 🌐 **Accessibility** works across devices
- 📊 **Search Functionality** find tools instantly
- 💰 **Monetized** with AdSense integration

## Tech Stack

- **Frontend:** Next.js 14+, TypeScript, Tailwind CSS
- **Deployment:** Vercel
- **Domain:** GoDaddy (custom domain)
- **SEO:** Google Search Console integration
- **Monetization:** Google AdSense

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/funney13valentine/toalltool.git
cd toalltool

# Install dependencies
npm install
# or
yarn install

# Run development server
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Building & Deployment

### Build for Production

```bash
npm run build
npm start
```

### Deploy to Vercel

This project is optimized for Vercel. Push to your repository and Vercel will automatically detect and deploy.

```bash
vercel deploy
```
